# HolidayHub – 2D Game

HolidayHub is a browser-based 2D game built using HTML, CSS, and JavaScript. The project focuses on interactive gameplay, simple mechanics, and a fun holiday-themed experience.

## About the Game

HolidayHub is a lightweight 2D game designed to run directly in the browser. Players explore the game world, interact with elements, and enjoy a playful holiday setting. The project was created to practice game development concepts using web technologies.

## Features

- 2D gameplay
- Keyboard-controlled player movement
- Interactive game elements
- Holiday-themed design
- Built using vanilla JavaScript
- Runs directly in the browser

## Controls

Action - Key
- Move Up - W / ↑
- Move Down - S / ↓
- Move Left - A / ←
- Move Right - D / →
- Shoot - Space
- Toggle Shoot - t

*(Controls may vary depending on implementation)*
